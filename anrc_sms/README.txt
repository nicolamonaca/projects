ANRC Safety Monitor System
==========================

* AUTHOR: Nicola Lamonaca
* DATE: June, 2011
* LANGUAGE: C#

* DESCRIPTION: ANRC SMS is a Windows Mobile application for the monitoring of radioactivity and temperature levels in a radiopharmaceutical firm. This was the project for the exam "Software Engineering", taught by professor Giuseppe Visaggio and got full marks with distinction.

You can find more info about ANRC SMS here (http://goo.gl/xEqZX) and here (http://goo.gl/5vRY9).
